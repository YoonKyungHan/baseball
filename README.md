# 🎮 실시간 멀티플레이어 야구게임

미국의 유명한 숫자 추측 게임을 웹 기반으로 구현한 실시간 멀티플레이어 게임입니다.

## 📋 게임 규칙

- **목표**: 상대방이 정한 4자리 숫자를 맞춰보세요!
- **제약**: 중복된 숫자는 사용할 수 없습니다 (0-9 중 서로 다른 4개 숫자)
- **결과**:
  - **스트라이크**: 숫자와 위치가 모두 맞음
  - **볼**: 숫자는 맞지만 위치가 틀림
  - **아웃**: 해당 숫자가 없음
- **승리**: 4 스트라이크 (HOME RUN!)를 달성하면 승리

## 🌟 주요 기능

- ⚡ **실시간 멀티플레이어**: 웹소켓을 이용한 실시간 1대1 게임
- 🎨 **아름다운 UI**: 현대적이고 직관적인 사용자 인터페이스
- 🎭 **임팩트 효과**: 스트라이크, 볼, 아웃에 따른 다양한 애니메이션 효과
- 🏆 **승리 효과**: 게임 승리 시 특별한 임팩트 효과
- 🏠 **로비 시스템**: 방 생성, 참가, 플레이어 수 표시
- 📱 **반응형 디자인**: 데스크톱과 모바일 모두 지원
- 🔄 **게임 히스토리**: 모든 추측과 결과를 실시간으로 기록

## 🚀 설치 및 실행

### 필요 조건
- Node.js (14.0.0 이상)
- npm

### 1. 의존성 설치
```bash
npm install
```

### 2. 서버 실행
```bash
# 프로덕션 모드
npm start

# 개발 모드 (자동 재시작)
npm run dev
```

### 3. 게임 접속
브라우저에서 `http://localhost:3000` 으로 접속하세요.

## 🎯 게임 플레이 방법

### 1단계: 로비 접속
- 닉네임을 입력하고 Enter를 누르세요
- 기존 방에 참가하거나 새 방을 만드세요

### 2단계: 방 입장
- 방에 2명이 모이면 게임이 시작됩니다
- 각자 4자리 비밀 숫자를 설정하세요 (중복 불가)

### 3단계: 게임 진행
- 턴이 오면 상대방의 숫자를 추측하세요
- 결과를 보고 다음 추측을 계획하세요
- 4 스트라이크를 달성하면 승리!

## 🌐 외부 접속 (ngrok 사용)

외부에서 접속할 수 있도록 하려면 ngrok를 사용하세요:

```bash
# ngrok 설치 (https://ngrok.com/)
# 서버 실행 후 다른 터미널에서:
ngrok http 3000
```

ngrok이 제공하는 URL을 친구들과 공유하여 함께 게임을 즐기세요!

## 🎨 특별 효과

- **스트라이크 효과**: 빨간색 바운스 애니메이션
- **볼 효과**: 노란색 펄스 애니메이션  
- **아웃 효과**: 보라색 흔들림 애니메이션
- **홈런 효과**: 무지개 그라디언트 + 지속적인 바운스
- **승리 효과**: 모달 창의 특별한 펄스 효과

## 🔧 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js
- **실시간 통신**: WebSocket (ws 라이브러리)
- **UI/UX**: CSS Grid, Flexbox, CSS Animations
- **상태 관리**: 클래스 기반 상태 관리
